# Workshop Mynds - IoT + AI 🤖

¡Bienvenido/a! En este repo vas a encontrar todo lo necesario para armar tu proyecto de IoT con Inteligencia Artificial del Workshop de Mynds.

## ¿Qué hay en este repo? 📁

### `/esp8266`
- Código para tu placa ESP8266
- Instrucciones para configurar la API de RecordAI

### `/serverless_function`
- Código Python para tu función Lambda en AWS
- Dos versiones:
  - Básica con Telegram (telegram_sin_ai.py)
  - Avanzada con AI para mensajes personalizados (telegram_con_ai.py)
- Librerías necesarias ([descargá el ZIP acá](https://github.com/juancereigido/mynds/raw/main/serverless_function/librerias.zip))
